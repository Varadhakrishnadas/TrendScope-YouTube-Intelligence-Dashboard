# TrendScope – YouTube Intelligence Dashboard

## Project Overview
TrendScope is a Python-based project that analyzes YouTube video data using the YouTube Data API.  
It performs engagement analysis, sentiment analysis, SEO scoring, and view prediction.

## Features
- Video engagement analysis (likes, comments, views)
- Sentiment analysis using TextBlob
- SEO score calculation
- Trend visualization using Matplotlib
- View forecasting using Machine Learning

## Tech Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn
- TensorFlow
- TextBlob
- YouTube Data API

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Add your YouTube API key in the notebook
4. Run trendscope_dashboard.ipynb
